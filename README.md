## Installation

```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## TradingView message format

```
{
    "ticker": "{{ticker}}",
    "direction": "{{strategy.order.action}}",
    "passphrase": "yourpassphrase"
}
```
